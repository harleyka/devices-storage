# Devices Storage

Simple app for devices storage, communicating with local API.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```


### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
